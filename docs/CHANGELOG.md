# 更新履歴 (Changelog)

## [0.2.2] - 2025-12-20

### Changed

- **スタイルシステム**: Tailwind CSSを完全に削除し、標準のCSS (Vanilla CSS) に移行しました。パフォーマンスとメンテナンス性が向上しました。
- **アクセシビリティ**: ドロップダウンメニューに `role="button"` 属性を追加し、アクセシビリティを改善しました。

### Fixed

- **タスク作成**: タスク入力フォームのレイアウト崩れを修正し、「Cancel」ボタンでフォームを閉じられるように改善しました。
- **タスクリスト**: タスク行のレイアウト崩れを修正しました。

### Added

- **テスト**: 主要コンポーネント（TaskInput, Sidebar, SubtaskList, ContextMenu, TaskList, CustomDropdown）のフロントエンドユニットテストを追加しました。
- **バックエンドテスト**: Rust側のタスク管理ロジックとファイルI/Oに対するユニットテストを追加しました。

## [0.2.1] - 2025-12-14

### Changed

- **UI改善**: タスク入力画面のStart DateとDue Dateの位置を入れ替え。

### Added

- **ガントチャート機能強化**:
  - **進捗表示**: サブタスクの消化状況に基づいた進捗率表示に対応。
  - **並び替え**: Start Date, Due Date, Nameによるソート機能を追加。
  - **依存関係**: タスク間の依存関係（Predecessors）の設定と可視化を実装。

### Fixed

- **ガントチャート**: 日付未設定タスクが存在する場合にクラッシュするバグを修正。
- **データ保存**: 依存関係（Dependencies）の変更が正しく保存されない問題を修正。

## [0.2.0] - 2025-12-14

### Added

- **ガントチャート**: タスクのスケジュールを視覚的に管理できるガントチャートビューを実装。タスクの依存関係や期間を直感的に把握可能。

### Changed

- **データ保存**: アプリケーションデータをSQLite DatabaseからJSONファイルベースに変更し、構成を簡素化。
- **ドキュメント**: README、仕様書、マニュアルをJSONストレージ構成に合わせて更新。

### Fixed

- **Markdown表示**: Markdownレンダリング時の箇条書きリスト表示を修正 (App.css)。

## [0.1.1] - 2025-12-13

### Fixed

- **Markdown表示**: Markdownレンダリング時のチェックリスト表示を修正。チェック有無が視覚的に明確（チェック時は着色）になるように改善。

## [0.1.0] - 2025-12-13

大規模なリファクタリングと新機能の追加を行いました。

### Added

- **データベース移行**: データ保存形式をJSONファイルからSQLite (SQLx) へ完全移行し、堅牢性とパフォーマンスを向上。
- **カレンダーイベント**: カレンダー上で独自のイベント（予定）を作成・管理する機能を追加。繰り返し設定や色分けが可能に。
- **Tailwind CSS**: UIスタイリングを既存のCSSからTailwind CSSへ移行し、モダンで統一感のあるデザインに刷新。
- **アーキテクチャ**: フロントエンドのコンポーネント構造を再設計・整理 (`src/components/` 以下の階層化)。

### Changed

- **アプリ名変更**: プロジェクト名を "tauri_todo" から "**Noruno Platform**" へ正式変更。
- **チェックボックス**: タスク完了時のチェックボックスサイズが変動するバグを修正。
- **設定画面**: メール設定やテーマ選択のUIを改善。

### Fixed

- 画像アセット (.jpg) を .pngに統一し、表示の不整合を解消。
- Linux (Arch系) でのビルドコマンドを追加・検証。
