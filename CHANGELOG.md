# 更新履歴 (Changelog)

## [0.2.0] - 2025-12-14

### Added

- **ガントチャート**: タスクのスケジュールを視覚的に管理できるガントチャートビューを実装。タスクの依存関係や期間を直感的に把握可能。

### Changed

- **データ保存**: アプリケーションデータを SQLite Database から JSON ファイルベースに変更し、構成を簡素化。
- **ドキュメント**: README、仕様書、マニュアルを JSON ストレージ構成に合わせて更新。

### Fixed

- **Markdown表示**: Markdownレンダリング時の箇条書きリスト表示を修正 (App.css)。

## [0.1.1] - 2025-12-13

### Fixed

- **Markdown表示**: Markdownレンダリング時のチェックリスト表示を修正。チェック有無が視覚的に明確（チェック時は着色）になるように改善。

## [0.1.0] - 2025-12-13

大規模なリファクタリングと新機能の追加を行いました。

### Added

- **データベース移行**: データ保存形式を JSON ファイルから SQLite (SQLx) へ完全移行し、堅牢性とパフォーマンスを向上。
- **カレンダーイベント**: カレンダー上で独自のイベント（予定）を作成・管理する機能を追加。繰り返し設定や色分けが可能に。
- **Tailwind CSS**: UI スタイリングを既存の CSS から Tailwind CSS へ移行し、モダンで統一感のあるデザインに刷新。
- **アーキテクチャ**: フロントエンドのコンポーネント構造を再設計・整理 (`src/components/` 以下の階層化)。

### Changed

- **アプリ名変更**: プロジェクト名を "tauri_todo" から "**Noruno Platform**" へ正式変更。
- **チェックボックス**: タスク完了時のチェックボックスサイズが変動するバグを修正。
- **設定画面**: メール設定やテーマ選択の UI を改善。

### Fixed

- 画像アセット (.jpg) を .png に統一し、表示の不整合を解消。
- Linux (Arch系) でのビルドコマンドを追加・検証。

---

## [0.0.5] - 2024-12-05

### Added

- **読書記録機能**: 書籍の登録、進捗管理、読書ノート、セッション記録機能を追加。
- **ダッシュボード**: タスクと読書の進捗を可視化するダッシュボード画面を追加。

### Changed

- サイドバーのデザインを更新し、新しい機能へのナビゲーションを追加。

---

## [0.0.4] - 2024-12-01

### Added

- **メモ機能**: Markdown 対応のメモ帳機能を追加。
  - フォルダによる階層管理
  - タグ付けと検索機能
- **自動保存**: メモの編集内容をリアルタイムに近い形で保存する仕組みを導入。

---

## [0.0.3] - 2024-11-30

### Added

- **サブタスク**: タスクを細分化して管理できるサブタスク機能を追加。
- **通知機能**: タスク期限前のメール通知・デスクトップ通知機能を実装。
- **設定**: 通知のタイミングを分単位で設定可能に。

---

## [0.0.2] - 2024-11-20

### Added

- **グループ機能**: タスクを任意のグループで分類する機能を追加。
- **カレンダービュー**: タスクを月間カレンダーで確認できるビューを追加。

---

## [0.0.1] - 2024-11-01

### Added

- **初期リリース**: 基本的なタスク管理機能（CRUD）の実装。
- **テーマ切り替え**: ライト/ダークモードを含む複数のカラーテーマに対応。
